# Cavelang
Simplified Python-like programming language based on caveman speak. Requires Python 3.14.0 and textX 4.3.0.
## Installation
```
git clone https://github.com/emnle04/CS420-Final-Project.git
```
## Hello World
```
say "Hello, world!"
```
## Usage
To run a program:
1. Open the repo folder.
2. Open a new terminal and enter the following:
```
python cavelang.py PROGRAM_NAME.ooga
```
Default program if unspecified is `program1.ooga`. Example programs are included in the source code.
